# Booking Platform API

Appointment and booking management system for service businesses.

## Tech Stack
- **Backend**: Java 17, Spring Boot 2.7, Spring Security + JWT, PostgreSQL 14
- **Frontend**: React 18, TypeScript

## Features
- Service listing and management
- Appointment booking with time slot selection
- Booking confirmation and cancellation
- JWT-based authentication
- Role-based access (Client, Provider, Admin)

## Setup
1. Configure PostgreSQL database
2. Run backend: `./mvnw spring-boot:run`
3. Run frontend: `cd frontend && npm start`
